LSD Log

The LSD Log window presents LSD messages to the user (e.g., errors, simulation run information, statistics about data).

Users can select the desired text by dragging over with the mouse. The selected text can be copied to the system clipboard by right-clicking and selecting the option Copy in the context menu or pressing Ctrl+C. The copied data can be pasted in any other program, as usual.

It is also possible to clear all the content of the Log window, by selecting the Clear option in the context menu, accessible by the right mouse button.
